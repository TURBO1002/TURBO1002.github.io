Google扩展程序的存储位置
2021-02-24 09:05:32 # Google # 扩展程序 # 前端

在本节中，我将介绍如何在自己的本地机器上找到已安装的Google浏览器插件的安装包。

1.存放位置查找

接下来，我将介绍如何在三种主流操作系统中去查找Google浏览器所安装的扩展程序的存放位置，请看查找方法：

  1. Mac OS

    ~/Library/Application Support/Google/Chrome/Default/Extensions

    图1 - Mac OS中插件的存放位置

  2. Linux（Ubuntu）

    ~/.config/google-chrome/Default/Extensions

  3. Windows

    # win7
C:\Users\用户名\AppData\Local\Google\Chrome\User Data\Default\Extensions

我们打开Google浏览器，并在地址栏中输入chrome://version/，可看到如图2所示界面：

图2 - Chrome信息

图2中「个人资料路径」下的 Extensions文件夹即为默认的扩展程序安装目录。
