优雅的在Pycharm查看JSON文件
2021-02-09 17:12:50 # python # Pycharm操作

当我们通过爬虫程序获取到来自服务端返回的JSON文件时，往往会在控制台中看到如图所示画面：

图1 - 控制台的json文件

这种格式不利于我们去获取到对应JSON文件中的值，因此我们需要将其格式化，这里我们提供三种方法：

  1. 使用「show as JSON」插件
  2. 使用第三方网站
  3. 使用Google插件

1. 使用「show as JSON」插件

为了我们更加方便直接的在Pycharm控制台中查看返回的JSON文件，我们需要安装「show as JSON」插件，请看安装步骤：

  1. 找到「Preferences」界面
图2 - Preferences信息

  1. 安装插件
    进入到「Preferences」界面之后，我们选中「Plugins」选项，并输入关键词「show As」，当查询到相关结果之后，我们点击「Install」按钮即可完成该插件的安装，如图所示：
    图3 - 安装插件

安装完成之后，我们就可以使用了，我们在控制台选中程序输出的JSON文件，并鼠标右键选中「show As JSON」，即可看到如图所示信息：

图4 - 插件使用方法

最终所得效果如图所示：

图5 - 效果展示

2. 使用第三方网站

我们进入到如下网址，http://www.bejson.com/

图6 - 第三方网站官网

接着，我们在输入框中输入需要转化的JSON文件并点击相应的按钮即可完成转换，如图所示：

图7 - 第三方网站官网转换结果展示
